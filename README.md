# POW – MERN Pet Accessories Store

A modern full stack eCommerce pet accessories store built with MongoDB, Express, React and Node.js (MERN).
The app includes product listings, categories, user authentication and a fully interactive shopping cart.
## Storefront (Frontend – React + Vite)
-Clean & modern UI
-Responsive hero banner with background image
-Pet categories:
     Popular Products
     Cat & Dog
     Bird
     Fish & Aquarium
     Trees & Indoor Plants
-Product grid layout with images
-Add to cart
 Real-time cart badge
-Cart drawer with:
    Quantity update
    Price calculation
    Remove items

## Environment Setup
Create .env inside the server/ folder:
MONGO_URI=your_mongodb_connection JWT_SECRET=your_secret_key
PORT=5050    

## Start Backend (server)
cd server
npm install
npm run dev

## Start Frontend (client)
cd client
npm install
npm run dev
